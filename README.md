# What Is For Dinner?

A responsive recipe generator web application built with HTML5, CSS3, JavaScript, and Bootstrap.

This project recreates a modern recipe website while focusing on JavaScript fundamentals such as object manipulation, arrays, DOM rendering, and random content generation.

## Live Demo

[https://what-is-for-dinner.vercel.app/
](https://what-is-for-dinner-gold.vercel.app/)
## Features

* Responsive design for mobile, tablet, and desktop devices
* Dynamic recipe generation
* Random recipe selection on button click
* Interactive recipe details
* Nutrition information display
* Ingredients and instructions tabs
* Chef tips section
* Bootstrap-powered layout and components
* Semantic HTML structure
* BEM naming convention

## JavaScript Highlights

The application stores recipe data inside an array of objects, where each recipe contains:

* General recipe information
* Ingredients
* Instructions
* Nutrition facts
* Chef tips

JavaScript is used to:

* Generate a random recipe
* Render recipe details dynamically
* Display ingredients lists
* Display preparation steps
* Display nutrition information
* Show conditional warning messages based on preparation time
* Update the entire recipe card without reloading the page
